§注意事项
1.  UnicornViewer的窗口不能最大化，也不能最小化，需要放置到一个不会被意外的弹出窗口遮住的地
    方，保持不要关闭；
2.  使用本脚本前，需要手动在UnicornViewer中生成一个pdf文件，保证所有弹出的窗口也在预期的区
    域内然后保持不要关闭；
3.  设置打印机的内容见截图；
    注意：
    A） 页面大小可以设置为：A4；
4.  参考命令行如下:
        ./pdg2pdf.sh 2>&1 | tee ./log.txt
