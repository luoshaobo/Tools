How to build:
1)  Go to the parent directory of kaola_accesslib_plugin.
2)  Execute:
        mkdir build__kaola_accesslib_plugin
        cd build__kaola_accesslib_plugin
        ../kaola_accesslib_plugin/qmake_dynamic_link.sh    # or: ../kaola_accesslib_plugin/qmake_static_link.sh
        make
       
       
How to Execute:
1)  Go to:
        build__kaola_accesslib_plugin/hmi_app_sample
2)  Execute:
        ./hmi_app_sample
