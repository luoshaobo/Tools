How to build:
1)  Go to the parent directory of qq_accesslib_plugin.
2)  Execute:
        mkdir build__qq_accesslib_plugin
        cd build__qq_accesslib_plugin
        ../qq_accesslib_plugin/qmake_dynamic_link.sh    # or: ../qq_accesslib_plugin/qmake_static_link.sh
        make
       
       
How to Execute:
1)  Go to:
        build__qq_accesslib_plugin/hmi_app_sample
2)  Execute:
        ./hmi_app_sample
